功能

为写长篇而生

不是云端笔记 App，也不是聊天框 + 一栏 prompt 的玩具。

📖

章节 + 卷两级结构

多卷分组，章节可拖拽归卷；字数统计、草稿 / 已完成状态、Ctrl+S 即存，一键导出整本 Markdown / TXT。

🤖

AI 多 Agent 协作

设定 Agent、剧情 Agent、文笔 Agent 并行跑，从不同维度给当前章节反馈。支持 DeepSeek、通义、GLM、Claude、Ollama 等 OpenAI 兼容 Provider。

👥

人物关系星图

圆形角色节点 + 彩色关系连线，6 种内置关系类型（敌对 / 爱慕 / 师徒 / 朋友 / 家人 / 其他）；右键改类型、双击改名、拖拽连线建关系。

📋

剧情时间轴

双轨道（明线 / 暗线）卡片式段落，每张卡片记录人物 / 地点 / 事件；章节或时间为轴，拖拽即排序，Delete 即删。

💾

本地优先

所有数据是你选的工作区文件夹下的 .md 和 .json。没有云、没有账户、没有锁定。复制走或 Git 管理都行。

🎨

原生跨平台

基于 Tauri 2，一份代码三个系统；浅色 / 深色主题一键切换；小说设定窗口可分离独立摆放。

开始

三步开写

从下载到第一个章节大概 2 分钟。

下载并安装

选你的系统对应安装包。Windows 首次启动 SmartScreen 拦截，点「更多信息」→「仍要运行」。

选一个工作区文件夹

启动后选任意空文件夹作为工作区，所有小说都会作为子目录存在那里。建议放在云盘外的本地路径。

配 AI Provider（可选）

在「⚙ 设置 → 模型」填一个 OpenAI 兼容 Key 和 Base URL，就能用 AI Agent。不配也能纯手写。

FAQ

常见问题

数据存在哪？能搬到别的电脑吗？
存在你启动时选的「工作区目录」里，全部是 .md 和 .json。整个目录直接打包拷贝过去就行，新机器装好应用启动时把工作区目录指过去即可。
支持哪些 AI Provider？
任何 OpenAI 兼容的 Chat Completion 接口：DeepSeek、通义千问、智谱 GLM、Moonshot、Anthropic 官方代理、本地 Ollama / LM Studio 都行。在「⚙ 设置 → 模型」里填 Base URL 和 API Key 即可。
需要联网吗？
章节编辑、设定、导出完全离线。只有调用 AI Agent 时才会向你配置的 Provider 发请求。
Windows 提示「未知发布者」？
自签名问题，不影响使用。SmartScreen 弹窗点「更多信息」→「仍要运行」即可。介意的话也可以从源码 npm run tauri build 自己编一个。
macOS 提示「无法打开，因为无法验证开发者」？
终端执行 xattr -dr com.apple.quarantine /Applications/诺娃写手.app 然后再双击启动。
免费吗？
当前阶段是。后续可能引入面向重度用户的付费功能，但章节编辑、AI 集成、设定编辑器这些核心创作能力会保持免费。
